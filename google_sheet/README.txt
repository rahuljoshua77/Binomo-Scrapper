TO RUN THIS PROGRAM YOU WILL NEED PYTHON INSTALLED ON YOUR WINDOWS/LINUX OPERATING SYSTEM-----------------------------------------

FOR WINDOWS:
	Python3.9 can be installed from the Microsoft Store

FOR LINUX:
	apt-get install python3

After installing python open cmd in the directory of the script.




INSTALLATION OF REQUIREMENTS----------------------------
pip3 install -r requirements.txt
